# Travis Barney's Orb Zone

Personal autobiographical slideshow built with Vue 3 and impress.js, deployed to Cloudflare Pages.

## Development

```bash
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run deploy
```

## Tech Stack

- Vue 3 (script setup SFCs)
- Vite
- [impress.js](https://impress.js.org/) - presentation framework
- Cloudflare Pages (deployment target)
